# Femto

A toy text editor based on antirez's "kilo" from [this tutorial](https://viewsourcecode.org/snaptoken/kilo/index.html).

## Installation

#### Build from source
```
git clone https://github.com/P1n3appl3/Femto.git
cd Femto
make
sudo make install
```
To uninstall simply `sudo make uninstall`
